### How to run locally

Make sure you have installed wails cli: https://wails.io/docs/gettingstarted/installation

INSTALL WAILS: `go install github.com/wailsapp/wails/v2/cmd/wails@v2.9.1`

BUILD: `wails build -clean -debug`

RUN WEB: `wails dev`
